Парисуддхи-упосатха (Чистота Упосатхи)

 

(i) Парисуддхи перед Сангхой
Заявление о своей чистоте перед Сангхой:


“Parisuddho ahaṃ bhante, parisuddho’ti maṃ saṅgho dhāretu.”
(«Я, почтенные бхиккху, полностью чист. Пусть Сангха считает меня чистым.»)
[ср. Vin,I,129]

 

(ii) Парисуддхи для 3 бхиккху
Для чтения Патимоккхи требуется как минимум четыре бхиккху. Если присутствуют только три бхиккху, то после выполнения предварительных обязанностей и общей исповеди один бхиккху произносит natti:


“Suṇantu me bhante, āyasmanto ajj’uposatho pannaraso, yadi āyasmantānaṃ pattakallaṃ, mayam annamannaṃ parisuddhi uposathaṃ kareyyāma.”
(«Пусть почтенные бхиккху услышат меня. Сегодня Упосатха, пятнадцатый день (двухнедельного периода). Если это подходит почтенным, давайте проведем Упосатху в полной чистоте между нами.»)

 

• Когда это 14-й день:
“pannaraso” → “catuddaso”

 

• Если объявляющий бхиккху является самым старшим:
“bhante” → “āvuso”

 

[ср. Vin,I,124]

 

Затем, начиная со старшего бхиккху:
“Parisuddho ahaṃ āvuso, parisuddho’ti maṃ dhāretha.”
(«Я, друзья, полностью чист. Понимайте, что я полностью чист.»)

 

Для каждого из двух младших бхиккху:
“āvuso” → “bhante”

 

(iii) Парисуддхи для 2 бхиккху


Пропустить natti. Старший бхиккху сначала заявляет о чистоте:
“Parisuddho ahaṃ āvuso, parisuddho’ti maṃ dhārehi.”
(«Я, друг, полностью чист. Понимай, что я полностью чист.»)

 

• Для младшего бхиккху:
“āvuso” → “bhante”
“dhārehi” → “dhāretha”
[ср. Vin,I,124–125]

 

(iv) Adhicchānuposatha (Для одного бхиккху)
Для бхиккху, который находится в одиночестве в день Упосатхи. После выполнения предварительных обязанностей он произносит:
“Ajja me uposatho.”
(«Сегодня для меня день Упосатхи.»)
[ср. Vin,I,125]
