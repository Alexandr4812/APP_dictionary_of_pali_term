PAVĀRAṆĀ (ПРИГЛАШЕНИЕ)

 

Для пяти или более бхиккху После предварительных обязанностей один бхиккху читает ñatti:

'Suṇātu me bhante saṅgho. Ajja pavāraṇā paṇṇarasī. Yadi saṅghassa pattakallaṃ, Saṅgho te-vācikaṃ pavāreyya.'

'Достопочтенные, пусть Сангха выслушает меня. Сегодня Pavāraṇā на пятнадцатый (день двухнедельного периода). Если Сангха готова, Сангха должна пригласить с тремя заявлениями.'

Vin.I.159

 

Когда это 14-й день:

'paṇṇarasī' → 'cātuddasī'

 

Если объявляющий бхиккху самый старший:

'bhante' → 'āvuso'

 

Если каждый бхиккху должен заявить свое приглашение дважды:

'te-vācikaṃ' → 'dve-vācikaṃ'

 

Если каждый бхиккху должен заявить свое приглашение один раз:

 'te-vācikaṃ' → 'eka-vācikaṃ'

 

Если бхиккху с равным стажем должны приглашать в унисон:

'Saṅgho te-vācikaṃ pavāreyya' → 'Saṅgho samāna-vassikaṃ pavāreyya'

'Сангха должна пригласить в манере равного стажа.'

 

После ñatti, если каждый бхиккху должен пригласить "трижды", то по порядку стажа:

'Saṅghaṃ bhante pavāremi. Diṭṭhena vā sutena vā parisaṅkāya vā, vadantu maṃ āyasmanto anukampaṃ upādāya. Passanto paṭikkarissāmi.

Dutiyam-pi bhante saṅghaṃ pavāremi. Diṭṭhena vā sutena vā parisaṅkāya vā, vadantu maṃ āyasmanto anukampaṃ upādāya. Passanto paṭikkarissāmi.

Tatiyam-pi bhante saṅghaṃ pavāremi Diṭṭhena vā sutena vā parisaṅkāya vā, vadantu maṃ āyasmanto anukampaṃ upādāya. Passanto paṭikkarissāmi.'

'Достопочтенные, я приглашаю Сангху к порицанию. Согласно тому, что было увидено, услышано или заподозрено, пусть достопочтенные наставляют меня из сострадания. Увидев это, я исправлюсь. Во второй раз... В третий раз...'

 

Для самого старшего бхиккху:

'Saṅghaṃ bhante' → 'Saṅghaṃ āvuso' 'Dutiyam-pi bhante' → 'Dutiyam-pi āvuso' 'Tatiyam-pi bhante' → 'Tatiyam-pi āvuso'

 

Для четырех или трех бхиккху Предварительные обязанности, затем ñatti:

'Suṇantu me āyasmanto, ajja pavāraṇā paṇṇarasī, yad'āyasmantānaṃ pattakallaṃ, mayaṃ aññamaññaṃ pavāreyyāma.'

'Достопочтенные, выслушайте меня. Сегодня pavāraṇā на 15-й (день двухнедельного периода). Если есть полная готовность достопочтенных, мы должны пригласить друг друга.'

Vin.I.162

 

Если есть три бхиккху:

 'āyasmanto' → 'āyasmantā'

 

Затем каждый бхиккху по порядку стажа:

'Ahaṃ bhante āyasmante pavāremi. Diṭṭhena vā sutena vā parisaṅkāya vā, vadantu maṃ āyasmanto anukampaṃ upādāya. Passanto paṭikkarissāmi.

Dutiyam-pi bhante āyasmante pavāremi. Diṭṭhena vā sutena vā parisaṅkāya vā, vadantu maṃ āyasmanto anukampaṃ upādāya. Passanto paṭikkarissāmi.

Tatiyam-pi bhante āyasmante pavāremi. Diṭṭhena vā sutena vā parisaṅkāya vā, vadantu maṃ āyasmanto anukampaṃ upādāya. Passanto paṭikkarissāmi.'

 

Для самого старшего бхиккху:

'bhante' → 'āvuso'

 

Если есть три бхиккху:

'āyasmanto' → 'āyasmantā'

 

Для двух бхиккху Предварительные обязанности, но без ñatti, затем каждый бхиккху по порядку стажа:

'Ahaṃ bhante āyasmantaṃ pavāremi. Diṭṭhena vā sutena vā parisaṅkāya vā, vadatu maṃ āyasmā anukampaṃ upādāya. Passanto paṭikkarissāmi.

Dutiyam-pi bhante āyasmantaṃ pavāremi. Diṭṭhena vā sutena vā parisaṅkāya vā, vadatu maṃ āyasmā anukampaṃ upādāya. Passanto paṭikkarissāmi.

Tatiyam-pi bhante āyasmantaṃ pavāremi. Diṭṭhena vā sutena vā parisaṅkāya vā, vadatu maṃ āyasmā anukampaṃ upādāya. Passanto paṭikkarissāmi.'

 

Для старшего бхиккху:

 'bhante' → 'āvuso'

 Vin.I.163

 

Для одного бхиккху Предварительные обязанности, затем:

'Ajja me pavāraṇā.'

 'Сегодня моя pavāraṇā.'

Vin.I.163

 

Pavāraṇā больного бхиккху

 'Pavāraṇaṃ dammi, pavāraṇaṃ me hara, mam'atthāya pavārehi.'

'Я даю свое приглашение. Передайте приглашение за меня. Пригласите от моего имени.'

Vin.I.161

 

Если больной бхиккху младший:

'hara' → 'haratha' 'pavārehi' → 'pavāretha'

 

Pavāraṇā больного бхиккху (например, 'Uttaro') передается на его месте в порядке стажа:

'Āyasmā bhante 'uttaro' gilāno saṅghaṃ pavāreti. Diṭṭhena vā sutena vā parisaṅkāya vā, vadantu taṃ āyasmanto anukampaṃ upādāya. Passanto paṭikkarissati.

Dutiyam-pi bhante āyasmā 'uttaro' gilāno… Passanto paṭikkarissati.

Tatiyam-pi bhante āyasmā 'uttaro' gilāno… Passanto paṭikkarissati.'

'Достопочтенные, достопочтенный 'Uttaro', который болен, делает приглашение Сангхе. С тем, что вы видели, слышали и подозревали, пусть все вы наставляете его из сострадания. Увидев это, он исправится.'

 

Если передающий бхиккху старше больного бхиккху:

'Āyasmā bhante 'uttaro'' → ''Uttaro' bhante bhikkhu'

Sp.V.1075
